Seletores CSS

Bem-vindo!

No CSS possuímos diversos seletores sendo esses:

  • Seletor por Tipo: busca elementos por uma tag HTML
  • Seletor por ID (#): busca elementos através de um id
  • Seletor por Classe (.): busca elementos através de uma classe
  • Seletor Universal (*): busca todos elementos
  • Seletor de Atributuo ({atrib}): busca todos elementos pelo atributo
